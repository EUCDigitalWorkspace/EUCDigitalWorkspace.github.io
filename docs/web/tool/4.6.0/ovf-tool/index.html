Open Virtualization Format (OVF) Tool

Overview

VMware OVF Tool is a command-line utility that helps you import and export OVF packages to and from many VMware products.

You can either install OVF Tool as an application, navigating to the top Download link, or unpack the Zip file for your platform and run it locally from the current folder.

The OVF Tool was released with vSphere 8.0 and 8.0 U1, but is backward compatible with earlier vSphere releases. For previous versions of this utility, see OVF Tool resources for vSphere 7 and before.

OVF is based on a DMTF specification. Background and details here at dmtf.org: Open Virtualization Format.
