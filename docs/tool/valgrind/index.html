Valgrind Tools, ESXi Version

Valgrind Tools, ESXi Version is an ESXi porting of Valgrind 3.7.0, a GPL (GNU General Public License)  licensed tool suite. It works with ESXi 5.5 and later versions. This suite comprises a number of tools for  memory debugging, memory leak detection, and cache and heap profiling.
